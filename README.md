## About Me

I am a **backend-focused Full Stack Software Engineer** with experience building marketplace platforms, scalable APIs, event-driven workflows, and modern web applications.

My work focuses on turning complex business requirements into maintainable modules, predictable data flows, secure authentication, and reliable background processing. I care about **clean architecture, concurrency, idempotency, database performance, and code that remains understandable as a product grows**.

I build backend systems with **Node.js, NestJS, Express.js, Python, Django, GraphQL, MongoDB, PostgreSQL, Redis, and RabbitMQ**, and frontend applications with **React, Next.js, TypeScript, Apollo Client, Redux Toolkit, and MUI**.
